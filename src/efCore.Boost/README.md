# efCore.Boost

Helpers and patterns to speed up EF Core development: async-only repositories, Unit-of-Work, OData helpers, EDM builders, bulk helpers, and query result models. Designed to be lightweight and VS Code friendly.

## Install
```bash
dotnet add package efCore.Boost
```

## Quick start
```csharp
// register your DbContext then your repos/UoW as usual
```

See the `samples/ConsoleSample` for a minimal setup and the `tests/efCore.Boost.Tests` for unit tests.
