# BoostX.Test

This test project uses **Testcontainers** to test data-access, seed data, and database objects such as Views and stored procedures.

**Note:** Testcontainers requires **Docker** to be running on the host machine.

This test is excellent for testing the integrity of the database objects and seed data.

For further details, see [TestContainers.md](./TestContainers.md).
